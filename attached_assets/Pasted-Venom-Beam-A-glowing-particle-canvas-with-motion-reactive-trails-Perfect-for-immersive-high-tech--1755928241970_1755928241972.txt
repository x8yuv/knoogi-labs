Venom Beam
A glowing particle canvas with motion-reactive trails. Perfect for immersive, high-tech hero sections.

API Reference
Venom Beam
A canvas of light and motion—crafted for bold digital intros.

Props
Property	Type	Default	Description
children	ReactNode	optional	
Content to render over the animated background. Usually includes headings, text, or buttons.
className	string	""	
Additional Tailwind CSS classes to apply to the overlay container holding the children.
